PROGRAMMING FINAL PROJECT: HARVESTHELP SL

1. Project Rationale
HarvestHelp SL is a GUI-based Python application designed to support smallholder farmers in Sierra Leone.
Many farmers estimate seed needs, input costs, yield and profit manually, which can lead to poor planning,
wasted resources and weak market decisions. The system provides a simple digital tool for entering farm
details and receiving clear farming projections before money is spent. It was selected under the Agriculture
Support System option because food production, input cost awareness and crop profitability are real
community concerns. The solution aligns with SDG 2: Zero Hunger by helping farmers improve planning,
productivity and household food security. It also supports SDG 8 through better income planning and local
economic activity.
2. System Explanation
The application collects farmer name, district, crop, land size, seed cost, fertilizer cost, labor cost, expected
market price and production risk. It calculates seed required, expected yield, adjusted market price, estimated
revenue, total cost, profit and decision status.
3. Structured Programming Application
The program uses constants, variables, dictionaries, functions, if/elif/else decisions, loops, meaningful names
and separated GUI and logic. It includes more than three user-defined functions: safe_float,
calculate_projection, format_currency, build_result_text and main.
4. Data, Privacy and Compliance
The system stores only basic farm planning records, avoids unnecessary sensitive data and exports records
locally as CSV. Data accessibility and interoperability are supported by the CSV format.
5. Open Source Standards
The project is prepared for GitHub publication under the MIT License with README.md, documentation,
screenshots and Python source files.
